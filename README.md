# 跨学科深度分析技能 (Deep Cross-Disciplinary Analysis) v4

## 架构理念

**v3问题**：按理论家/学科组织，20+理论家堆叠，无法再扩展。
**v4解法**：按**认知操作**组织——你想做什么？学科是工具箱里的工具，按需调用。

## 六个认知操作

| 操作 | 核心问题 | 调用学科 |
|------|---------|---------|
| OP-1 机制拆解 | 底层怎么运作？ | 脑科学 · 认知科学 · 行为科学 · 精神分析 · 医学生理 |
| OP-2 结构透视 | 什么系统在塑造它？ | 社会学 · 经济学 · 政治学 · 博弈论 · 投资思维 · 法律 |
| OP-3 语言解剖 | 语言揭示了什么？ | 语言学 · 修辞学 · 叙事学 · 文学批评 |
| OP-4 逻辑审计 | 哪里有漏洞/反直觉？ | 形式逻辑 · 认知偏差 · 反直觉模式目录 |
| OP-5 比较映射 | 结构性差异是什么？ | 比较文学 · 知识史 · 跨文化研究 |
| OP-6 实践萃取 | 当下能做什么？ | 决策科学 · 美学 · 哲学 · 行为设计 |

## 核心升级点（vs v3）

- **架构重构**：理论家清单 → 认知操作系统，可无限扩展学科工具而不改变架构
- **新增学科**：脑科学(奖赏回路/杏仁核/前额叶/镜像神经元/可塑性)、认知科学(卡尼曼双系统/前景理论)、行为科学(习惯回路/助推)、医学生理(HPA轴/心身症状)、投资思维(期权性/反脆弱/尾部风险)、美学(崇高感/审美距离)、决策科学(预mortem/贝叶斯更新)、哲学(存在主义/斯多葛)
- **对抗性验证**：从8项打勾式checklist升级为两遍式对抗性审计
- **反直觉检测**：新增7种反直觉模式目录
- **知识卡片输出**：可直接进Obsidian，对齐CLIFF框架
- **适用范围扩大**：从"分析一本书"扩展到"分析一本书或一个议题"
