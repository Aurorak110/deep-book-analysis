# 深度书籍分析技能 (Deep Book Analysis) v3.1

## 概述
多维度深度书籍分析技能，面向有深度阅读习惯的中文读者。

## 能力架构

### 五维主体分析
1. **心理分析 (Psyche)** — 拉康 / 吉拉尔 / 弗洛伊德 / 荣格 / 温尼科特 / 鲍尔比
2. **制度、权力与社会 (Structure & Society)** — 福柯 / 阿伦特 / 布尔迪厄 / 韦伯 / 马克思 / 涂尔干 / 戈夫曼 / 齐美尔 / 法律对比
3. **经济视角 (Economy)** — 政治经济学(马克思四部专著) / 博弈论 / 路径依赖 / 婚姻经济学
4. **语言与认知 (Language)** — 平克 / 莱考夫 / 萨丕尔-沃尔夫 / 巴赫金
5. **当代警醒 (Mirror)** — 模式识别 / 决策节点 / 结构性陷阱的当代版本

### 三个附加模块
- **A. 比较分析** — 同作者纵向 + 跨作者横向（含作者配对数据库）
- **B. 逻辑审计** — 角色推理谬误识别 + 分析自审
- **C. 法律对比** — 跨法域法律制度结构性比较

### 文学批评工具箱
形式主义 / 叙事学 / 新历史主义 / 后殖民批评 / 读者反应批评

### 字数预算（v3.1 新增）
单维度深挖 / 五维标准 / 五维+1附加 / 全模块完整 — 四档自动选择，避免输出过载

### 自动校准机制
8项输出质量检查，每次输出前内部执行
